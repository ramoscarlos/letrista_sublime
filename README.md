# Letrista

## Atajos de teclado

`Ctl + /` - Comentar líneas.  

## Snippets

`ss` - Genera una sección de cuatro líneas.  
`ll` - Genera el inicio con rima indeterminada y cero sílabas.  
`lll` - Esqueleto de letra con título, dos versos, un puente, y tres coros.
